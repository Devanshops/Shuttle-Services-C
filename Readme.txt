Shuttle service is a shuttle booking and management system.
It comes with easy maintenance and usability for the user-end and  the admin-end both.

*************************************************************************************************

Steps to run file in code blocks:

1.) Extract all the files from .zip file.

2.) Create an empty project.

3.) Add all files (including .dat files from .zip file) to the project 
    and store them in the same project folder.

4.) Run and debugg the files.

5.) Execute the main file (proj.c) of ADT.

    And code will be executed.

***************************************************************************************************

Steps to run using gcc on Ubuntu(or any other linux):

1.) Extract all the files from .zip file.

2.) Open terminal and set current directory to the folder containing extracted files.

3.) Compile porj.c and func.c together as:
	"gcc proj.c func.c" (without quotes).

4.) Execute using "./a.out" (without quotes).

	And code will be executed.

**************************************************************************************************

Additional Info.

Default login credentials are "admin" and "admin".

	cut and paste password.dat in the same directory as forget_pass.c 
	run forget_pass.c.
